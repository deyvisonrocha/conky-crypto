# conky-crypto
Conky with my Portfolio cryptocurrency

<img src='crypto.png'>

## Installation
- Clone repo `git clone https://github.com/deyvisonrocha/conky-crypto.git ~/.conky/crypto`
- Install Python dependencies:
  - babel.numbers
- Use Conky Manager to autostart when login
